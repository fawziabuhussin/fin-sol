# App Privacy (App Store Connect questionnaire)

Complete at: App Store → App Privacy → Get Started

## Data collected

| Data type | Collected | Linked to user | Tracking | Purpose |
|-----------|-----------|----------------|----------|---------|
| Email Address | Yes | Yes | No | App Functionality, Account |
| Name | Optional | Yes | No | App Functionality |
| Financial Info | Yes | Yes | No | App Functionality |
| User Content | Yes | Yes | No | App Functionality |
| Product Interaction | Optional | Yes | No | Analytics (if you add analytics later, update) |
| Crash Data | Optional | No | No | App Functionality |

## Notes
- Financial info = transactions, salary, savings amounts the user enters
- User content = transaction descriptions, payslip PDFs, project notes
- Data is NOT used for advertising or sold to data brokers
- Privacy Policy URL: https://fin-sol-pied.vercel.app/privacy
